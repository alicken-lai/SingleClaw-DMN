# meeting_minutes_to_report – Examples

## Sample Input (`examples/sample_input.json`)

```json
{
  "meeting_title": "Q3 Product Planning",
  "date": "2024-07-15",
  "attendees": ["Alice", "Bob", "Carol"],
  "raw_notes": "Discussed roadmap for Q3. Alice to handle onboarding redesign by end of July. Bob will set up staging environment. Carol raised concern about API latency – needs investigation. Decision: delay v2.0 launch to August. Next meeting in two weeks."
}
```

## Expected Output Highlights

- Summary paragraph covering roadmap and v2.0 delay decision
- Action items table with Alice (onboarding), Bob (staging), Carol (API investigation)
- Clean professional Markdown report
